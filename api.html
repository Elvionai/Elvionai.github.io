Quick Start

Get up and running in minutes. Here's your first API call to Seraphina. 

curl -X POST "https://web-production-9a18.up.railway.app/api/developer/seraphina/chat" \
  -H "Authorization: Bearer YOUR_ELVION_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "message": "Tell me about quantum computing",
    "stream": false
  }'
import requests

url = "https://web-production-9a18.up.railway.app/api/developer/seraphina/chat"
headers = {
    "Authorization": "Bearer YOUR_ELVION_API_KEY",
    "Content-Type": "application/json"
}
data = {
    "message": "Tell me about quantum computing",
    "stream": False
}

response = requests.post(url, headers=headers, json=data)
print(response.json())
const response = await fetch(
  "https://web-production-9a18.up.railway.app/api/developer/seraphina/chat",
  {
    method: "POST",
    headers: {
      "Authorization": "Bearer YOUR_ELVION_API_KEY",
      "Content-Type": "application/json"
    },
    body: JSON.stringify({
      message: "Tell me about quantum computing",
      stream: false
    })
  }
);

const data = await response.json();
console.log(data);

Authentication

Elvion AI uses API key authentication. Include your key in the Authorization header as a Bearer token. 

Authorization: Bearer YOUR_ELVION_API_KEY
Security Tip: Never expose your API key in client-side code or public repositories. Store it securely as an environment variable.
