Whether you are building a conversational assistant, a creative writing tool, or a digital art generator, the Elvion API provides the endpoints you need with the speed and reliability you demand.

Base URL

All API requests must be directed to:

https://web-production-9a18.up.railway.app

Authentication

Security is paramount. All access to Elvion AI's protected endpoints requires a valid API Key. This key allows us to track usage and manage your credit consumption.

Authorization Header

Pass your API key in the standard Authorization HTTP header using the Bearer scheme.

headers.config 
Authorization: Bearer YOUR_ELVION_API_KEY

Error Handling

Our API speaks clearly. When things go wrong, we return standard HTTP status codes along with a JSON object describing the issue.

CodeMeaningDescription
200OKThe request was successful.
401UnauthorizedYour API Key is missing or invalid.
402Payment RequiredYou have run out of credits. Recharge to continue.
500Server ErrorInternal misconfiguration (usually missing Environment Variables).

Seraphina Chat

POST /api/developer/seraphina/chat 5 Credits

Seraphina is the Nonchalant Guru. She is engineered for deep research, factual analysis, and structured reasoning. She utilizes external knowledge bases (Perplexity) to provide up-to-date information.

Parameters

FieldTypeRequiredDescription
message string Yes The user's query. E.g., "Analyze the impact of quantum computing on encryption."
stream boolean No Defaults to true. Set to false for a single JSON response object.
history array No Context window. List of {role: "user"|"assistant", content: "..."}. 
curl -X POST "https://web-production-9a18.up.railway.app/api/developer/seraphina/chat" \
-H "Authorization: Bearer YOUR_ELVION_API_KEY" \
-H "Content-Type: application/json" \
-d '{
  "message": "Explain the Fermi Paradox.",
  "stream": false
}'
import requests

url = "https://web-production-9a18.up.railway.app/api/developer/seraphina/chat"
headers = {
    "Authorization": "Bearer YOUR_ELVION_API_KEY",
    "Content-Type": "application/json"
}
data = {
    "message": "Explain the Fermi Paradox.",
    "stream": False
}

response = requests.post(url, headers=headers, json=data)
print(response.json())
const axios = require('axios');

const url = "https://web-production-9a18.up.railway.app/api/developer/seraphina/chat";
const headers = {
    "Authorization": "Bearer YOUR_ELVION_API_KEY",

axios.post(url, {

Image Generation

POST /api/developer/generate-image 15 Credits

Turn text into visual reality. This endpoint accesses our high-performance diffusion models. The result is returned as a Base64 Encoded String, which you can easily render in any frontend.

Parameters

FieldTypeRequiredDescription
promptstringYesDetailed description of the scene.
modelstringNoOptions: v6, photoreal, anime_core, movie. Default: v6.
widthintNoImage width (e.g., 1024).
heightintNoImage height (e.g., 1024).

Response Format

The API returns a JSON object: { "image_b64": "..." }. You must decode this string or use it as a data source (data:image/png;base64,...) to display the image.
