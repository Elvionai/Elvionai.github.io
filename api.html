Authentication

Accessing Elvion AI's protected endpoints requires a valid API Key, transmitted as a Bearer Token in your request headers. Think of it as your digital key to our mystical realms of intelligence.

Any attempt to commune with the API without this key, or with an invalid one, will be met with a 401 Unauthorized response. 

Authorization: Bearer YOUR_ELVION_API_KEY

Each invocation of a protected endpoint consumes credits from your account. Should your credit balance be insufficient for the requested operation, the API will respond with a 402 Payment Required status, urging you to replenish your digital coffers.

Seraphina Chat

Unleash Seraphina, the Nonchalant Guru, Elvion AI's premier conversational model designed for profound research, meticulous analysis, and empirically grounded responses. Perfect for applications requiring deep factual inquiry and structured insights. This endpoint supports both real-time streaming and complete responses.

POST /api/developer/seraphina/chat 5 Credits

Request Body

Field Type Description
message string Required. The user's latest query or prompt to Seraphina.
history array of objects Optional. An array representing past conversational turns. Each object must contain "role" ("user" or "assistant") and "content" (string).
temperature float Optional. Controls the creativity and randomness of Seraphina's response. A value between 0.0 (more deterministic) and 1.0 (more creative). Defaults to 0.7.
stream boolean Optional. Set to true to receive the response in a continuous stream, or false for a single, complete JSON response. Defaults to true. For server-side applications, false is often easier to handle. 
curl -X POST "https://web-production-9a18.up.railway.app/api/developer/seraphina/chat" \
-H "Authorization: Bearer YOUR_ELVION_API_KEY" \
-H "Content-Type: application/json" \
-d '{
  "message": "What are the latest advancements in AI-driven drug discovery?",
  "stream": false
}'
import requests
import json

API_URL = "https://web-production-9a18.up.railway.app/api/developer/seraphina/chat"
ELVION_API_KEY = "YOUR_ELVION_API_KEY"

headers = {
    "Authorization": f"Bearer {ELVION_API_KEY}",
    "Content-Type": "application/json"
}

payload = {
    "message": "What are the latest advancements in AI-driven drug discovery?",
    "history": [
        {
            "role": "user",
            "content": "Give me a brief overview of machine learning."
        },
        {
            "role": "assistant",
            "content": "Machine learning is a field of AI that focuses on enabling systems to learn from data..."
        }
    ],
    "stream": False # Set to false to get a complete JSON response
}

response = requests.post(API_URL, headers=headers, json=payload)

if response.status_code == 200:
    print(json.dumps(response.json(), indent=2))
else:
    print(f"Error: {response.status_code}")
    print(response.text)

Generate Image

Bring your visions to life with Elvion AI's powerful image generation service. Describe any scene, character, or concept, and our models will craft a stunning visual representation. Choose from various artistic models to achieve your desired aesthetic.

POST /api/developer/generate-image 15 Credits

Request Body

Field Type Description
prompt string Required. A vivid and detailed description of the image you wish to conjure. Be as imaginative as possible!
model string Optional. Specifies the desired artistic model. Available options include: "v6" (powerful for high-res images), "photoreal" (for extreme realism), "movie" (cinematic images with filters), "anime_core" (base anime style), "illustration" (versatile illustration), "realistic" (photorealistic detail), "deepanime" (legacy deep anime), "animelegacy" (legacy classic anime), "dreamcore" (surreal aesthetic). Defaults to "v6".
width integer Optional. The desired width of the generated image in pixels. If omitted, an optimal default for the selected model is used.
height integer Optional. The desired height of the generated image in pixels. If omitted, an optimal default for the selected model is used.
creative boolean Optional. Set to true for more artistic and varied outputs. Defaults to false.
fastMode boolean Optional. Set to true to prioritize speed over absolute quality (may reduce processing time). Defaults to false.
denseMode boolean Optional. Set to true for denser, more detailed image generation (may increase processing time). Defaults to false.

The endpoint responds with a JSON object containing an image_b64 field, which holds the base64-encoded PNG image data. This data can be directly embedded into HTML or decoded for storage.

curl -X POST "https://web-production-9a18.up.railway.app/api/developer/generate-image" \
-H "Authorization: Bearer YOUR_ELVION_API_KEY" \
-H "Content-Type: application/json" \
-d '{
  "prompt": "A majestic lion with a crown made of stars, photorealistic, 4k, cinematic lighting",
  "model": "photoreal"
}'
import requests
import base64
from PIL import Image
from io import BytesIO

API_URL = "https://web-production-9a18.up.railway.app/api/developer/generate-image"
ELVION_API_KEY = "YOUR_ELVION_API_KEY"

headers = {
    "Authorization": f"Bearer {ELVION_API_KEY}",
    "Content-Type": "application/json"
}

payload = {
    "prompt": "A majestic lion with a crown made of stars, photorealistic, 4k, cinematic lighting",
    "model": "photoreal",
    "width": 768,
    "height": 1024
}

response = requests.post(API_URL, headers=headers, json=payload)

if response.status_code == 200:
    data = response.json()
    image_b64 = data.get("image_b64")
    
    if image_b64:
        # Decode and save the image
        image_data = base64.b64decode(image_b64)
        image = Image.open(BytesIO(image_data))
        image.save("elvion_generated_image.png")
        print("Image successfully generated and saved as elvion_generated_image.png")
    else:
        print("No image data received.")
else:
    print(f"Error: {response.status_code}")
    print(response.text)
