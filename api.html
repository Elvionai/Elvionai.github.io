Quick Start

One call. One response. Fully Elvionic. Use these snippets to verify your key and feel the API.

The minimal call: send a POST request to Seraphina with a message and your Bearer token. The examples below are fully copy‚Äëpaste friendly. 

curl -X POST "https://web-production-9a18.up.railway.app/api/developer/seraphina/chat" \
  -H "Authorization: Bearer YOUR_ELVION_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "message": "Give me 3 startup ideas in education technology.",
    "stream": false
  }'

Authentication

One key. One header. All the power. Every protected request carries your Elvion key as a Bearer token.

Include your key in the Authorization header: 

Authorization: Bearer YOUR_ELVION_API_KEY

Keys look like elv_sec_xxxxxxxxx. Treat them as secrets: keep them on the server side, in environment variables, and never embed them in public front‚Äëend code or Git repositories.

Base URL

All endpoints orbit a single base: a fast, HTTPS‚Äëonly gateway hosted on modern infra.

https://web-production-9a18.up.railway.app

Append each endpoint path to this base, for example /api/developer/seraphina/chat, /api/developer/bobby/chat, or /api/developer/generate-image.

Seraphina Chat

Seraphina is the nonchalant guru: analytical, unhurried, but razor‚Äësharp. Use her when you want thorough reasoning, research, and structured answers.

POST /api/developer/seraphina/chat 5 Credits

Send a question in message, optionally pass history and temperature, and get a JSON response with Seraphina‚Äôs answer and usage data.

Request Body

Field Type Description
message Required string Main user prompt or question.
history Optional array Array of past turns, each like {"role": "user"|"assistant", "content": "..."}.
temperature Optional float Randomness (0.0‚Äì1.0). Lower = more focused, higher = more creative. Default 0.7.
stream Optional boolean Set false for a single JSON response. Default true (streaming).

import requests, base64
from io import BytesIO
from PIL import Image

API_URL = "https://web-production-9a18.up.railway.app/api/developer/generate-image"
API_KEY = "YOUR_ELVION_API_KEY"

resp = requests.post(
    API_URL,
    headers={
        "Authorization": f"Bearer {API_KEY}",
        "Content-Type": "application/json",
    },
    json={
        "prompt": "A neon-blue city floating in space, stars everywhere, ultra detailed, 4k",
        "model": "movie",
        "width": 1024,
        "height": 1024,
    },
)

data = resp.json()
img_b64 = data["image_b64"]
img_bytes = base64.b64decode(img_b64)

img = Image.open(BytesIO(img_bytes))
img.save("elvion_space_city.png")

                            


                            


const API_URL = "https://web-production-9a18.up.railway.app/api/developer/generate-image";
const API_KEY = "YOUR_ELVION_API_KEY";

async function generateImage(prompt) {
  const res = await fetch(API_URL, {
    method: "POST",
    headers: {
      "Authorization": `Bearer ${API_KEY}`,
      "Content-Type": "application/json",
    },
    body: JSON.stringify({ prompt, model: "movie" }),
  });

  const data = await res.json();
  const img = document.createElement("img");
  img.src = `data:image/png;base64,${data.image_b64}`;
  document.body.appendChild(img);
}

generateImage("A neon-blue city floating in space, stars everywhere, ultra detailed, 4k");

Backend Integration Snippets

                    

                        Drop these into your backend to proxy Elvion safely (no keys on the front‚Äëend).
                    

                


                

                    
Node.js (Express)

                    

                        

                            
                        

                        

                            
                            


import express from "express";
import fetch from "node-fetch";

const app = express();
app.use(express.json());

const ELVION_API_KEY = process.env.ELVION_API_KEY;
const SERAPHINA_URL =
  "https://web-production-9a18.up.railway.app/api/developer/seraphina/chat";

app.post("/api/seraphina", async (req, res) => {
  try {
    const { message } = req.body;

    const upstream = await fetch(SERAPHINA_URL, {
      method: "POST",
      headers: {
        "Authorization": `Bearer ${ELVION_API_KEY}`,
        "Content-Type": "application/json",
      },
      body: JSON.stringify({ message, stream: false }),
    });

    const data = await upstream.json();
    res.status(upstream.status).json(data);
  } catch (err) {
    console.error(err);
    res.status(500).json({ error: "Upstream error" });
  }
});

app.listen(3000, () => {
  console.log("Proxy listening on http://localhost:3000");
});
