Authentication

Base URL for all requests:

https://web-production-9a18.up.railway.app

Pass your API key in the Authorization header as a Bearer token.

Authorization: Bearer elv_sec_your_api_key_here

Generate Video

Available endpoints (cost 32 credits/sec):

  • POST /api/developer/generate-video-from-text
  • POST /api/developer/generate-video-from-image
  • POST /api/developer/generate-video-from-frames
  • POST /api/developer/generate-video-subject
These endpoints return a task_id. You must poll /api/video/status?task_id=... to get the final download URL.
