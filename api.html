Getting Started

Welcome to the Elvion AI Developer API, your gateway to a universe of intelligent creation. Our API provides powerful, direct access to our suite of AI models, including advanced chat, hyper-realistic image generation, and more.

To begin your journey, you'll need an API key. You can generate and manage your keys from your developer dashboard.

Forge your API Key by signing up on the Developer Portal.

Base URL

All API endpoints are relative to the following base URL:

https://web-production-9a18.up.railway.app

Authentication

The Elvion AI API uses Bearer Token authentication for its protected endpoints. You must include your secret API key in the Authorization header with every request.

Requests without a valid API key will be rejected with a 401 Unauthorized status. API usage is metered in credits. If your balance is insufficient, the request will fail with a 402 Payment Required status. 

Authorization: Bearer YOUR_API_KEY

Seraphina Chat

Access Seraphina, our advanced AI assistant optimized for deep research, professional analysis, and factual, web-grounded responses. This endpoint is ideal for applications requiring accuracy and depth.

POST /api/developer/seraphina/chat 5 Credits

Request Body

FieldTypeDescription
messagestringRequired. The user's latest message or prompt.
historyarrayOptional. An array of previous messages for conversation context.
temperaturefloatOptional. Controls randomness (0.0 to 1.0). Defaults to 0.7.
streambooleanOptional. Set to false to receive the full response at once. Defaults to true.
 
curl -X POST "https://web-production-9a18.up.railway.app/api/developer/seraphina/chat" \
-H "Authorization: Bearer YOUR_API_KEY" \
-H "Content-Type: application/json" \
-d '{
  "message": "What are the latest advancements in AI-driven drug discovery?",
  "stream": false
}'
import requests
import json

API_URL = "https://web-production-9a18.up.railway.app/api/developer/seraphina/chat"
API_KEY = "YOUR_API_KEY"
headers = {"Authorization": f"Bearer {API_KEY}", "Content-Type": "application/json"}
payload = {
    "message": "What are the latest advancements in AI-driven drug discovery?",
    "stream": False
}

response = requests.post(API_URL, headers=headers, json=payload)
print(json.dumps(response.json(), indent=2))

Generate Image

Conjure stunning visuals from text. Our image generation endpoint translates your imagination into high-resolution reality, offering a variety of models for different artistic styles.

POST /api/developer/generate-image 15 Credits

Request Body

FieldTypeDescription
promptstringRequired. A detailed description of the image to generate.
modelstringOptional. Specifies the model: "v6" (general), "photoreal" (realism), "anime_core" (anime style). Defaults to "v6".
widthintegerOptional. The width of the image. Defaults to an optimal size for the model.
heightintegerOptional. The height of the image. Defaults to an optimal size for the model.
 
curl -X POST "https://web-production-9a18.up.railway.app/api/developer/generate-image" \
-H "Authorization: Bearer YOUR_API_KEY" \
-H "Content-Type: application/json" \
-d '{
  "prompt": "A majestic lion with a crown made of stars, photorealistic, 4k",
  "model": "photoreal"
}'
import requests
import base64
from io import BytesIO
from PIL import Image

API_URL = "https://web-production-9a18.up.railway.app/api/developer/generate-image"
API_KEY = "YOUR_API_KEY"
headers = {"Authorization": f"Bearer {API_KEY}", "Content-Type": "application/json"}
payload = {
    "prompt": "A majestic lion with a crown made of stars, photorealistic, 4k",
    "model": "photoreal"
}

response = requests.post(API_URL, headers=headers, json=payload)
if response.status_code == 200:
    image_b64 = response.json().get("image_b64")
    image_data = base64.b64decode(image_b64)
    image = Image.open(BytesIO(image_data))
    image.save("generated_image.png")
    print("Image saved as generated_image.png")
else:
    print(f"Error: {response.text}")
const API_URL = "https://web-production-9a18.up.railway.app/api/developer/generate-image";
const API_KEY = "YOUR_API_KEY";
const data = {
    prompt: "A majestic lion with a crown made of stars, photorealistic, 4k",
    model: "photoreal",
};

async function generateImage() {
    const response = await fetch(API_URL, {
        method: 'POST',
        headers: {
            'Authorization': `Bearer ${API_KEY}`,
            'Content-Type': 'application/json'
        },
        body: JSON.stringify(data)
    });
    const result = await response.json();
    const imageUrl = `data:image/png;base64,${result.image_b64}`;
    // You can now use imageUrl as the src for an  tag.
    console.log("Image received successfully!");
}
generateImage();
