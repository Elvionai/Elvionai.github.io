Elvion AI / Developer Protocol

Welcome to the source. You are accessing the developer interface for Elvion AI's cognitive engines. This documentation outlines the protocols to integrate our advanced neural models into your application matrix.

Access Required: Obtain your API credentials from the Developer Portal.
BASE_URL https://web-production-9a18.up.railway.app

Authentication

Security is paramount. All requests to the neural core must be signed with your Elvion API Key. This key acts as your identity and credit wallet within the system. 

Authorization: Bearer ELV-xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx

* 401 Unauthorized: Invalid or missing key.
* 402 Payment Required: Insufficient credits in your developer wallet.

Invoke Seraphina

Seraphina (The Nonchalant Guru) is optimized for high-fidelity research, structural analysis, and fact-grounded responses. Use this endpoint for complex queries requiring accuracy.

POST /api/developer/seraphina/chat 5 Credits / Request
ParameterTypeProtocol
message string Required. The input query for the model.
history array Optional. Contextual memory: [{ "role": "user", "content": "..." }]
stream boolean Defaults to true. Set false for JSON response. 
curl -X POST "https://web-production-9a18.up.railway.app/api/developer/seraphina/chat" \
-H "Authorization: Bearer YOUR_API_KEY" \
-H "Content-Type: application/json" \
-d '{
  "message": "Analyze the impact of quantum computing on cryptography.",
  "stream": false
}'
import requests

url = "https://web-production-9a18.up.railway.app/api/developer/seraphina/chat"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}
data = {
    "message": "Analyze the impact of quantum computing on cryptography.",
    "stream": False
}

response = requests.post(url, headers=headers, json=data)
print(response.json())
const response = await fetch("https://web-production-9a18.up.railway.app/api/developer/seraphina/chat", {
  method: "POST",
  headers: {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
  },
  body: JSON.stringify({
    message: "Analyze the impact of quantum computing on cryptography.",
    stream: false
  })
});

console.log(await response.json());

Fabricate Reality

Elvion's visual synthesis engine. Convert textual descriptions into high-fidelity imagery. Supports multiple stylistic models for diverse artistic output.

POST /api/developer/generate-image 15 Credits / Request
ParameterTypeProtocol
prompt string Required. Detailed visual description.
model string Optional. v6, photoreal, movie, anime_core.
width/height integer Optional. Dimensions of the output canvas. 
curl -X POST "https://web-production-9a18.up.railway.app/api/developer/generate-image" \
-H "Authorization: Bearer YOUR_API_KEY" \
-H "Content-Type: application/json" \
-d '{
  "prompt": "Cyberpunk city with neon rain, 8k resolution",
  "model": "photoreal"
}'
import requests

url = "https://web-production-9a18.up.railway.app/api/developer/generate-image"
