HOW IT WORKS
Create
Go to Create, describe your website in detail. Choose V1 (standard) or V2 (premium). AI generates and deploys it live.
Edit & Add Pages
Open any project, use the AI Edit tab. You can click "+ Add Page" to make the AI build navigation menus and multiple hidden sections to act like pages!
Manual Code
Switch to the Code tab to edit HTML directly. Click the eye to preview. Your changes are saved locally.
Bobby AI
Tap the purple robot button anytime for expert web design advice and help crafting prompts.
Images
Upload photos to postimg.cc, copy the direct link, add it to your project. AI can embed them in your site.
