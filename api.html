Base URL

All Elvion AI endpoints are rooted at the following base URL; prepend it to each path documented below.

Example: Seraphina Chat is POST /api/developer/seraphina/chat, which resolves to https://web-production-9a18.up.railway.app/api/developer/seraphina/chat.

Format: JSON over HTTPS HTTP status codes for error reporting Language‑agnostic usage

Authentication

Authentication uses standard bearer tokens in the Authorization HTTP header. Treat your Elvion API key like a password—never embed it in front‑end code, public repos, or client‑side logs.

All protected endpoints require the header below on each request, with your secret key substituted in. 

Authorization: Bearer YOUR_ELVION_API_KEY
Content-Type: application/json
Credits & billing. Each call to a protected endpoint consumes credits from your Elvion account. If your balance is insufficient, you will receive 402 Payment Required instead of a normal response.

Seraphina Chat

Seraphina is Elvion’s research‑driven, nonchalant guru—built for fact‑dense exploration, structured analysis, and grounded reasoning. Use this endpoint when you need precise answers and multi‑step thinking, not casual banter.

Ideal use cases include technical explanations, strategy ideation, complex “why/how” questions, and analytical workflows that must stay anchored to reality.

POST /api/developer/seraphina/chat 5 Credits

Request and response bodies are JSON. Set stream to false to receive a single payload instead of a stream.

Request body

Field Type Required Description
message string Yes Latest user query or prompt to give Seraphina context for this turn.
history array of objects No Conversation history. Each item has role ("user" or "assistant") and content (string). Send previous turns to keep the dialogue continuous.
temperature number No Controls creativity vs. determinism, typically between 0.0 and 1.0. Lower values are more focused; higher values are more exploratory. Defaults to 0.7.
stream boolean No When false, the API responds with a single JSON object. When true, responses may be streamed chunk‑by‑chunk (depending on your integration).

Code examples

curl -X POST "https://web-production-9a18.up.railway.app/api/developer/seraphina/chat" \
  -H "Authorization: Bearer YOUR_ELVION_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "message": "Summarize the latest trends in AI-driven drug discovery.",
    "temperature": 0.4,
    "stream": false
  }'

Bobby Chat

Bobby is the friendly, expressive side of Elvion—excellent for creative writing, code sketches, drafts, and emotionally rich conversations. Reach for Bobby when you want personality and imagination.

Typical scenarios include story crafting, UX copy, prototype code, email drafts, and any conversational experience that should feel warm rather than clinical.

POST /api/developer/bobby/chat 2 Credits

Bobby accepts a concise prompt plus an optional system‑level instruction to steer tone, style, or behavior.

Request body

Field Type Required Description
prompt string Yes Main user prompt. Ask Bobby to write, code, brainstorm, or chat.
systemPrompt string No Extra instruction that defines Bobby’s personality or constraints, for example “You are a senior Python engineer who explains things step‑by‑step”.

Generate Image

Turn text into visuals with Elvion’s image generator. Describe the mood, style, and subject; the engine returns a base64‑encoded PNG that you can display, store, or post‑process in your product.

You can tune the visual style with the model field and optionally specify width and height for finer layout control.

POST /api/developer/generate-image 15 Credits

The response includes an image_b64 field containing a base64 string representation of a PNG image.

Request body

Field Type Required Description
prompt string Yes Rich textual description of the image you want: subject, style, lighting, mood, composition, and any other constraints.
model string No Image model name, for example "v6", "photoreal", "movie", "anime_core", "illustration", "realistic", "deepanime", or "dreamcore". Defaults to "v6".
width integer No Target width in pixels. If omitted, the backend chooses a sensible default based on the selected model.
height integer No Target height in pixels. As with width, leaving this out lets the backend apply its default aspect ratio.

const API_URL = "https://web-production-9a18.up.railway.app/api/developer/generate-image";
const ELVION_API_KEY = "YOUR_ELVION_API_KEY";

async function generateImage(prompt) {
  const res = await fetch(API_URL, {
    method: "POST",
    headers: {
      Authorization: `Bearer ${ELVION_API_KEY}`,
      "Content-Type": "application/json",
    },
    body: JSON.stringify({
      prompt,
      model: "v6",
      width: 768,
      height: 768,
    }),
  });

  if (!res.ok) {
    const err = await res.text();
    throw new Error(`HTTP ${res.status}: ${err}`);
  }

  const data = await res.json();
  const imgB64 = data.image_b64;
  if (!imgB64) {
    console.error("No image_b64 in response", data);
    return;
  }

  const url = `data:image/png;base64,${imgB64}`;
  const img = document.createElement("img");
  img.src = url;
  img.alt = "Elvion generated image";
  document.body.appendChild(img);
}

generateImage("A cyberpunk alley with holographic signs and gentle rain, ultra detailed.");
