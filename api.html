Authentication

Secure your connection to the grid.

All requests must be authenticated via the Authorization header using the Bearer scheme. Your credits are deducted in real-time based on usage.

Header Format 
Authorization: Bearer YOUR_ELVION_API_KEY
Credit Policy: Insufficient funds will result in a 402 Payment Required status. Keep your energy reserves charged.

Seraphina Chat

Deep research, factual analysis, and nonchalant wisdom.

POST /api/developer/seraphina/chat 5 Credits

Seraphina is engineered for complex inquiries requiring web access and empirical data. She minimizes hallucination and maximizes depth.

curl -X POST "https://web-production-9a18.up.railway.app/api/developer/seraphina/chat" \
-H "Authorization: Bearer YOUR_KEY" \
-H "Content-Type: application/json" \
-d '{
  "message": "Analyze the atmospheric composition of Neptune.",
  "stream": false
}'
import requests

url = "https://web-production-9a18.up.railway.app/api/developer/seraphina/chat"
headers = {
    "Authorization": "Bearer YOUR_KEY",
    "Content-Type": "application/json"
}
data = {
    "message": "Analyze the atmospheric composition of Neptune.",
    "stream": False
}

response = requests.post(url, headers=headers, json=data)
print(response.json())
const url = "https://web-production-9a18.up.railway.app/api/developer/seraphina/chat";

fetch(url, {
    method: 'POST',
    headers: {
        'Authorization': 'Bearer YOUR_KEY',
        'Content-Type': 'application/json'
    },
    body: JSON.stringify({
        message: "Analyze the atmospheric composition of Neptune.",
        stream: false
    })
})
.then(res => res.json())
.then(console.log);

Generate Image

Manifest visions into reality. 4K visuals from pure text.

POST /api/developer/generate-image 15 Credits

Parameters:

  • prompt (Required): Description of the image.
  • model: "v6", "photoreal", "anime_core", "cyberpunk". (Default: v6)
  • width/height: Dimensions of the output.

-H "Authorization: Bearer YOUR_KEY" \
-H "Content-Type: application/json" \

  "prompt": "A futuristic city on Mars, neon lights, 8k resolution",
