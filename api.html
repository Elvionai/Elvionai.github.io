Elvion AI API

Welcome to the Elvion AI Developer Documentation. Our API empowers developers to integrate state-of-the-art AI capabilities directly into their applications. Leverage Seraphina for deep reasoning and research, Bobby for friendly conversational logic, and our Visual Engine for high-fidelity image generation.

Base URL: https://web-production-9a18.up.railway.app

Authentication

The API uses Bearer Token authentication. You must include your API key in the header of every request requiring developer access.

To obtain an API key, visit the Developer Dashboard.

Authorization: Bearer YOUR_API_KEY

Credits & Pricing

The API operates on a Pay-As-You-Go credit system. Each endpoint consumes a specific amount of credits per successful request.

Service Cost (Credits)
Seraphina Chat (Research) 5
Bobby Chat (Conversation) 2
Image Generation 15
POST /api/developer/seraphina/chat

Seraphina Chat

Access "The Nonchalant Guru". Powered by Perplexity Sonar Pro, Seraphina specializes in deep research, web-grounding, and complex reasoning.

Request Body

  • message (string, required) The user's prompt or question.
  • history (array, optional) Previous conversation messages `[{role: "user", content: "..."}]`.
  • temperature (float, optional) Creativity level (0.0 - 1.0). Default: 0.7.
  • stream (boolean, optional) If true, returns Server-Sent Events. Default: true.
POST /api/developer/bobby/chat

Bobby Chat

Access "Bobby". A lightweight, thoughtful, and friendly intelligence. Optimized for quick interactions, coding assistance, and creative writing.

Request Body

  • prompt (string, required) The input prompt for Bobby.
  • systemPrompt (string, optional) Override the default system behavior.
POST /api/developer/generate-image

Generate Image

Create stunning visuals using Thena v6 or MiniMax models. Supports photorealism, anime styles, and complex text rendering.

Request Body

  • prompt (string, required) Detailed description of the image.
  • model (string) Options: `v6` (default), `photoreal`, `anime_core`, `movie`.
  • use_minimax (boolean) Set `true` to use the MiniMax engine instead of Thena.
  • width (int, optional) Image width (e.g., 1024).
  • height (int, optional) Image height (e.g., 1024). 
curl -X POST "https://web-production-9a18.up.railway.app/api/developer/seraphina/chat" \
-H "Authorization: Bearer YOUR_API_KEY" \
-H "Content-Type: application/json" \
-d '{
    "message": "Explain quantum entanglement",
    "temperature": 0.7,
    "stream": false
}'

import requests
import base64

url = "https://web-production-9a18.up.railway.app/api/developer/generate-image"
payload = {
    "prompt": "A futuristic city in Lagos, neon lights, 8k",
    "model": "v6",
    "width": 1024,
    "height": 1024
}
headers = { "Authorization": "Bearer YOUR_API_KEY" }

response = requests.post(url, json=payload, headers=headers)
data = response.json()

# Save the base64 string as an image
if "image_b64" in data:
    with open("output.png", "wb") as f:
        f.write(base64.b64decode(data["image_b64"]))
