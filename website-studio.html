HOW IT WORKS
1. CreateGo to Create → describe your website in detail → pick V1 or V2 → Generate. AI builds and deploys it live with a URL.
2. Edit EndlesslyOpen any project → AI Edit tab. Use quick edits or type custom instructions. Every edit redeploys and updates your live site. All changes saved forever.
3. Add PagesIn editor, tap "+ Add Page" quick edit. AI builds navigation with multiple sections styled like separate pages (Single Page App).
4. Code EditorSwitch to Code tab to edit HTML directly. Changes auto-save and update preview.
5. Bobby AITap the purple robot button anytime for expert web design advice, prompt help, or to plan your site.
6. ImagesUpload to postimg.cc → copy direct link → paste in Images tab. Click "Insert in Site" to embed via AI.
7. Permanent StorageAll your projects are saved to your account in the cloud. They will never be lost. Sign in on any device to access them.
