Authentication Matrix

Access to Elvion AI's core functionalities is a privileged operation. Every request must be authenticated using your unique API Key, transmitted within the Authorization header as a Bearer Token. This token is your digital signature across the void.

An invalid or missing token will result in a connection severance, flagged with a 401 Unauthorized response code.

Each successful transaction with our API expends energy credits from your account. An insufficient credit balance will trigger a 402 Payment Required protocol, halting further operations until your reserves are replenished. Maintain your energy levels to ensure uninterrupted service.

Anomaly Detection

The universe of code is not without its anomalies. Our API communicates the status of your requests using standard HTTP codes. A 2xx response signals a successful operation. A 4xx response indicates a localized issue with your request—a malformed query, invalid parameters, or authentication failure. A 5xx response signals a systemic anomaly on our end, a rare event in the stable Elvion AI core.

Common Anomaly: 400 Bad Request

This often indicates a missing or incorrectly formatted field in your JSON payload. Scrutinize your request body against the endpoint's required parameters. Ensure all data types match the specifications.

Common Anomaly: 500 Internal Server Error

If you encounter this, it signifies a problem within our core systems. While we engineer for resilience, cosmic events can occur. Please wait a moment and retry your request. If the anomaly persists, our automated systems are likely already working to resolve it.

import requests
import base64

API_URL = "https://web-production-9a18.up.railway.app/api/developer/generate-image"
API_KEY = "YOUR_ELVION_API_KEY"

headers = {"Authorization": f"Bearer {API_KEY}", "Content-Type": "application/json"}
payload = {
    "prompt": "A cybernetic dragon soaring through a nebula, fiery exhaust, neon blue circuits, 8k, cinematic",
    "model": "photoreal"
}
response = requests.post(API_URL, headers=headers, json=payload)

if response.status_code == 200:
    image_b64 = response.json().get("image_b64")
    with open("generated_image.png", "wb") as f:
        f.write(base64.b64decode(image_b64))
    print("Image saved as generated_image.png")
else:
    print(f"Error: {response.status_code}")

const API_URL = "https://web-production-9a18.up.railway.app/api/developer/generate-image";

        },
        body: JSON.stringify(data)
    });
    const result = await response.json();
    const imageUrl = `data:image/png;base64,${result.image_b64}`;
