Introduction

Welcome to the Elvion AI Developer API! Our platform provides powerful, creative, and intelligent endpoints to integrate cutting-edge AI features into your applications. From advanced conversational models to state-of-the-art image generation, the Elvion API is your partner in building the next generation of intelligent software.

This documentation provides a comprehensive overview of our available endpoints, authentication methods, and billing system.

Base URL

All API endpoints are relative to the following base URL:

https://web-production-9a18.up.railway.app

Authentication

The Elvion AI Developer API uses a simple Bearer Token authentication model. You must include an API key in the Authorization header of every request to the protected developer endpoints.

Getting Your API Key

1. First, you need an Elvion AI account. If you don't have one, please sign up.
2. Log in to the Developer Dashboard.
3. On the dashboard, you can generate, view, and revoke your API keys. Generating your first API key costs 100 credits.

Making Authenticated Requests

To authenticate your API requests, include an Authorization header with the value Bearer YOUR_API_KEY.

Important: Keep your API keys secure. Do not share them in publicly accessible areas such as GitHub, client-side code, etc.

Billing & Credits

The Elvion Developer API operates on a pay-as-you-go credit system. Each endpoint has a specific credit cost per call, which is automatically deducted from your account balance.

How Credits Work

  • You can purchase credits through your Developer Dashboard.
  • Each API call to a protected endpoint consumes a set number of credits.
  • If your credit balance is insufficient to cover the cost of an API call, the request will fail with a 402 Payment Required status code.
  • You can check your current balance at any time via the dashboard or the Get Keys & Balance API endpoint.

Endpoint Costs

Endpoint Cost per Call
Seraphina Chat 5 Credits
Bobby Chat 2 Credits
Generate Image 15 Credits
Generate API Key 100 Credits

API Keys & Payments

Create API Key

POST /api/user/create-api-key

Generates a new developer API key for your account. This action requires authentication via a Firebase Token (obtained upon logging into the dashboard) and costs 100 credits.

Headers

HeaderDescription
X-Firebase-Token Your Firebase authentication token. Required

Responses

200 OK

{
  "api_key": "elvn_...",
  "message": "API key created successfully!"
}

402 Payment Required

{
  "detail": "Insufficient credits. Cost: 100. Balance: 50."
}

Get Keys & Balance

GET /api/user/keys-and-balance

Retrieves a list of all your active API keys and your current account credit balance. Requires authentication via a Firebase Token.

Headers

HeaderDescription
X-Firebase-Token Your Firebase authentication token. Required

Responses

200 OK

{
  "keys": [
    {
      "key": "elvn_...key1",
      "created_at": "2026-01-05T18:00:00Z",
      "is_active": true
    },
    {
      "key": "elvn_...key2",
      "created_at": "2026-01-04T12:00:00Z",
      "is_active": true
    }
  ],
  "balance": 1500
}

Revoke API Key

POST /api/user/revoke-api-key

Deactivates one of your API keys, preventing it from being used for any future requests. This action is irreversible. Requires authentication via a Firebase Token.

Headers

HeaderDescription
X-Firebase-Token Your Firebase authentication token. Required

Request Body

ParameterDescription
api_key The full API key you wish to revoke. string Required

Responses

200 OK

{
  "message": "API key has been successfully revoked."
}

Fund Wallet

POST /api/payments/initiate

Initiates a payment transaction to add credits to your account. This endpoint returns a transaction reference, which you can use to complete the payment with our provider (Squad). The actual payment and verification happen on the frontend. Requires authentication via a Firebase Token.

Headers

HeaderDescription
X-Firebase-Token Your Firebase authentication token. Required

Request Body

ParameterDescription
amount The amount in Naira (NGN) you wish to add. The minimum amount is 100. This amount will be converted to credits. integer Required

Responses

200 OK

{
  "transaction_ref": "elvion-..."
}

Developer Endpoints

Seraphina Chat

POST /api/developer/seraphina/chat 5 credits

Access Seraphina, our premium AI model specializing in deep research, professional analysis, and expert reasoning. Powered by Perplexity, Seraphina provides web-grounded, factual, and cited responses. This endpoint supports streaming.

Request Body

Parameter Description
message The user's latest message or prompt. string Required
history An array of previous messages to provide conversation context. Must alternate between "user" and "assistant" roles. array of objects
temperature Controls randomness. Higher values (e.g., 0.9) are more creative, while lower values (e.g., 0.2) are more deterministic. float (0.0 to 1.0)
stream Set to true to receive a server-sent event (SSE) stream of tokens. Set to false for a single JSON response. Defaults to true. boolean

History Object Structure

{
  "role": "user | assistant",
  "content": "The text of the message."
}

Responses

200 OK (Non-streamed)

{
  "reply": "This is Seraphina's detailed response...",
  "citations": [
    { "title": "Source Title", "url": "https://example.com" }
  ],
  "usage": {
    "prompt_tokens": 50,
    "completion_tokens": 150,
    "total_tokens": 200
  }
}

200 OK (Streamed)

The response will be a stream of server-sent events. Your client should be configured to handle text/event-stream.

Generate Image

POST /api/developer/generate-image 15 credits

Create stunning, high-quality images from a text prompt. This versatile endpoint can use either our primary provider (Thena) or our secondary provider (MiniMax).

Request Body

Parameter Description
prompt A detailed description of the image you want to generate. string Required
model (Thena only) The specific model to use. See available models below. Defaults to v6. string
use_minimax Set to true to use the MiniMax provider instead of Thena. Defaults to false. boolean
creative (Thena only) Enables a more creative generation mode. boolean
width (Thena only) The width of the image. If not provided, an optimal width for the selected model is used. integer
height (Thena only) The height of the image. If not provided, an optimal height for the selected model is used. integer

Available Thena Models

photoreal, movie, v6, anime_core, illustration, realistic

Responses

200 OK

{
  "image_b64": "iVBORw0KGgoAAAANSUhEUgA... (base64 encoded PNG image)"
}
