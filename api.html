Introduction

Welcome to the Elvion AI Developer API. Our platform provides developers with programmatic access to state-of-the-art conversational and generative AI models. Integrate intelligent chat, deep research capabilities, and powerful image generation directly into your applications with ease.

This interactive documentation provides everything you need to get started, including authentication guides, detailed endpoint references, and copy-paste code examples. All API access is managed through a credit-based system from your Developer Dashboard.

Authentication

All requests to the Elvion AI API must be authenticated using an API key. You can generate and manage your keys from the Developer Dashboard. To authenticate, include your API key in the Authorization header as a Bearer token.

POST /api/developer/bobby/chat 2 Credits

Access the Bobby model for fast and cost-effective conversational AI. Ideal for general-purpose chat and creative text generation where real-time web access is not required.

Request Body

PARAMETERTYPEDESCRIPTION
promptstringRequired. The message you want to send to the model.
systemPromptstringOptional. A custom system prompt to guide the model's personality.

Responses

200 OKSuccessful Response

Returns a JSON object with the model's reply. See the code panel for an example.

402 Payment RequiredInsufficient Credits

Returned if your credit balance is below the cost of the request.

POST /api/developer/seraphina/chat 5 Credits

Access Seraphina, our premium, web-grounded research assistant. Provides accurate, up-to-date information and supports streaming for real-time responses.

Request Body

PARAMETERTYPEDESCRIPTION
messagestringRequired. The message or question for Seraphina.
historyarrayOptional. An array of previous messages for context. E.g., [{"role": "user", "content": "Hi"}, ...]
streambooleanOptional. Defaults to true for streaming. Set to false for a single response.
POST /api/developer/generate-image 15 Credits

Generate high-quality images from a text prompt using either our default (Thena) or alternative (MiniMax) generation models.

Request Body

PARAMETERTYPEDESCRIPTION
promptstringRequired. A detailed description of the desired image.
modelstringOptional. For Thena, e.g., "photoreal", "v6". Defaults to "v6".
use_minimaxbooleanOptional. Set to true to use the MiniMax model. Defaults to false.

Responses

200 OKSuccessful Response

Returns a JSON object containing the generated image as a Base64-encoded string: {"image_b64": "..."}

Error Handling

The API uses standard HTTP status codes. Failed requests will return a JSON object with a detail key containing a human-readable error message.

CODEMEANING
401 UnauthorizedAuthentication failed. Your API key is invalid or missing.
402 Payment RequiredYour account has insufficient credits for this action.
422 Unprocessable EntityYour request body is malformed or missing required fields.
5xx Server ErrorAn unexpected error occurred on our end.

BOBBY CHAT REQUEST

import requests

API_URL = "https://web-production-9a18.up.railway.app/api/developer/bobby/chat"
API_KEY = "YOUR_API_KEY"

headers = {
    "Authorization": f"Bearer {API_KEY}",
    "Content-Type": "application/json"
}
data = {
    "prompt": "Hello, who are you?"
}
response = requests.post(API_URL, headers=headers, json=data)
print(response.json())

IMAGE GENERATION REQUEST

import requests
import base64
from PIL import Image
from io import BytesIO

API_URL = "https://web-production-9a18.up.railway.app/api/developer/generate-image"
API_KEY = "YOUR_API_KEY"

headers = { "Authorization": f"Bearer {API_KEY}", "Content-Type": "application/json" }
data = {
    "prompt": "A cat wearing a tiny wizard hat, studio lighting",
    "model": "v6"
}
response = requests.post(API_URL, headers=headers, json=data)

if response.status_code == 200:
    b64_string = response.json()['image_b64']
    img_data = base64.b64decode(b64_string)
    img = Image.open(BytesIO(img_data))
    img.save("generated_image.png")
    print("Image saved as generated_image.png")
else:
    print(response.text)

BOBBY CHAT REQUEST

// Using Node.js with node-fetch
async function callBobby() {
    const response = await fetch('https://.../bobby/chat', {
        method: 'POST',
        headers: {
            'Authorization': `Bearer ${'YOUR_API_KEY'}`,
            'Content-Type': 'application/json'
        },
        body: JSON.stringify({ prompt: 'Hello!' })
    });
    console.log(await response.json());
}
callBobby();

IMAGE GENERATION REQUEST

// Using Node.js with node-fetch and fs
const fs = require('fs');

async function generateImage() {
    const response = await fetch('https://.../generate-image', {
        method: 'POST',
        headers: {
            'Authorization': `Bearer ${'YOUR_API_KEY'}`,
            'Content-Type': 'application/json'
        },
        body: JSON.stringify({ 
            prompt: 'A cat wearing a tiny wizard hat' 
        })
    });
    const data = await response.json();
    fs.writeFileSync('image.png', data.image_b64, 'base64');
    console.log('Image saved!');
}
generateImage();
