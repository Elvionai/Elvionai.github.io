Getting Started

Welcome to the Elvion AI Developer API! Our API provides powerful access to our suite of AI models, including advanced chat, image generation, and more. This documentation will guide you through authenticating and using our various endpoints.

To start using the API, you'll need an API key. You can generate and manage your keys from your developer dashboard.

Get your API Key by signing up on the Developer Login Page.

Base URL

All API endpoints are relative to the following base URL:

https://web-production-9a18.up.railway.app

Authentication

The Elvion AI API uses Bearer Token authentication. You must include your API key in the Authorization header for all requests to protected endpoints.

Requests without a valid API key will fail with a 401 Unauthorized status code. 

Authorization: Bearer YOUR_API_KEY

API usage is metered in credits. Each call to a protected endpoint will deduct a specific amount of credits from your account balance. If your balance is insufficient, the request will fail with a 402 Payment Required status code.

Seraphina Chat

Access Seraphina, our advanced AI assistant optimized for deep research, professional analysis, and factual, web-grounded responses. This endpoint supports streaming for real-time interaction.

POST /api/developer/seraphina/chat 5 Credits

Request Body

Field Type Description
message string Required. The user's latest message or prompt.
history array Optional. An array of previous messages to provide conversation context. Each object should have role ("user" or "assistant") and content (string).
temperature float Optional. Controls randomness. A value between 0.0 and 1.0. Defaults to 0.7.
stream boolean Optional. Whether to stream the response back. Defaults to true. For a developer API, you might prefer false to get the full response at once. 
curl -X POST "https://web-production-9a18.up.railway.app/api/developer/seraphina/chat" \
-H "Authorization: Bearer YOUR_API_KEY" \
-H "Content-Type: application/json" \
-d '{
  "message": "What are the latest advancements in AI-driven drug discovery?",
  "stream": false
}'
import requests
import json

API_URL = "https://web-production-9a18.up.railway.app/api/developer/seraphina/chat"
API_KEY = "YOUR_API_KEY"

headers = {
    "Authorization": f"Bearer {API_KEY}",
    "Content-Type": "application/json"
}

payload = {
    "message": "What are the latest advancements in AI-driven drug discovery?",
    "history": [
        {
            "role": "user",
            "content": "Give me a brief overview of machine learning."
        },
        {
            "role": "assistant",
            "content": "Machine learning is a field of AI that..."
        }
    ],
    "stream": False # Set to false to get a complete JSON response
}

response = requests.post(API_URL, headers=headers, json=payload)

if response.status_code == 200:
    print(json.dumps(response.json(), indent=2))
else:
    print(f"Error: {response.status_code}")
    print(response.text)

const API_URL = "https://web-production-9a18.up.railway.app/api/developer/seraphina/chat";
const API_KEY = "YOUR_API_KEY";

const data = {
    message: "What are the latest advancements in AI-driven drug discovery?",

})
.then(response => {
    if (!response.ok) {
        throw new Error(`HTTP error! status: ${response.status}`);

})
.then(data => {
    console.log(JSON.stringify(data, null, 2));

    console.error('Error:', error);
});

Generate Image

Create stunning images from a text prompt using our powerful generation models. You can choose between different providers and models for various styles.

POST /api/developer/generate-image 15 Credits

Request Body

Field Type Description
prompt string Required. A detailed description of the image you want to generate.
use_minimax boolean Optional. Set to true to use the MiniMax provider. Defaults to false (uses Thena).
model string Optional. Only for Thena (when use_minimax is false). Specifies the model, e.g., "v6", "photoreal", "anime_core". Defaults to "v6".
width integer Optional. The width of the generated image. If omitted, an optimal default for the selected model is used.
height integer Optional. The height of the generated image. If omitted, an optimal default for the selected model is used.

The endpoint returns a JSON object with an image_b64 field containing the base64-encoded PNG image data.

curl -X POST "https://web-production-9a18.up.railway.app/api/developer/generate-image" \
-H "Authorization: Bearer YOUR_API_KEY" \
-H "Content-Type: application/json" \
-d '{
  "prompt": "A majestic lion with a crown made of stars, photorealistic, 4k",
  "model": "photoreal"
}'
import requests
import base64
from PIL import Image
from io import BytesIO

API_URL = "https://web-production-9a18.up.railway.app/api/developer/generate-image"
API_KEY = "YOUR_API_KEY"

headers = {
    "Authorization": f"Bearer {API_KEY}",
    "Content-Type": "application/json"
}

payload = {
    "prompt": "A majestic lion with a crown made of stars, photorealistic, 4k",
    "model": "photoreal"
}

response = requests.post(API_URL, headers=headers, json=payload)

if response.status_code == 200:
    data = response.json()
    image_b64 = data.get("image_b64")
    
    # To save the image
    image_data = base64.b64decode(image_b64)
    image = Image.open(BytesIO(image_data))
    image.save("generated_image.png")
    print("Image saved as generated_image.png")
else:
    print(f"Error: {response.status_code}")
    print(response.text)
