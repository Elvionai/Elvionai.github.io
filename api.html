Seraphina Chat

Access the "Nonchalant Guru." Powered by Perplexity Sonar Pro, Seraphina provides deep, web-grounded answers with citations. She is analytical, precise, and transparent.

POST /api/developer/seraphina/chat

Request Body

Parameter Type Required Description
message string Yes The user's query or prompt.
history array No List of previous messages [{"role": "user", "content": "..."}].
temperature float No Creativity level (0.1 - 1.0). Default: 0.7.
stream boolean No If true, returns Server-Sent Events (SSE). Default: true. 
curl -X POST "https://web-production-9a18.up.railway.app/api/developer/seraphina/chat" \
-H "Authorization: Bearer YOUR_API_KEY" \
-H "Content-Type: application/json" \
-d '{
    "message": "Explain quantum computing with citations",
    "history": [],
    "temperature": 0.7,
    "stream": false
}'
import requests

url = "https://web-production-9a18.up.railway.app/api/developer/seraphina/chat"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}
data = {
    "message": "Explain quantum computing with citations",
    "stream": False 
}

response = requests.post(url, json=data, headers=headers)
print(response.json())

Bobby Chat

Bobby is the steady, dependable intelligence. Powered by Thena Llama, he is excellent for coding, creative writing evaluation, and logical breakdowns. He is not web-connected.

POST /api/developer/bobby/chat

Request Body

Parameter Type Required Description
prompt string Yes The input text/question.
systemPrompt string No Override the default "Bobby" persona. 
curl -X POST "https://web-production-9a18.up.railway.app/api/developer/bobby/chat" \
-H "Authorization: Bearer YOUR_API_KEY" \
-H "Content-Type: application/json" \
-d '{
    "prompt": "Write a python function to calculate Fibonacci numbers."
}'
import requests

response = requests.post(
    "https://web-production-9a18.up.railway.app/api/developer/bobby/chat",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={"prompt": "Write a python function to calculate Fibonacci numbers."}
)
print(response.json())

Generate Image

Unleash visual creativity using Thena v6 or MiniMax models. Create photorealistic, anime, or 3D art instantly.

POST /api/developer/generate-image

Request Body

Parameter Type Default Description
prompt string Required Detailed visual description.
model string v6 Options: v6, photoreal, anime_core, movie.
use_minimax boolean false Set true to use MiniMax engine instead of Thena.
fastMode boolean false Trade detail for speed (Thena only). 
curl -X POST "https://web-production-9a18.up.railway.app/api/developer/generate-image" \
-H "Authorization: Bearer YOUR_API_KEY" \
-H "Content-Type: application/json" \
-d '{
    "prompt": "A futuristic Lagos city skyline with flying cars, neon lights, ultra realistic, 8k",
    "model": "photoreal",
    "width": 1024,
    "height": 1024
}'
import requests
import base64

payload = {
    "prompt": "A cyberpunk samurai, neon rain, detailed armor",
    "model": "v6",
    "fastMode": True
}
headers = {"Authorization": "Bearer YOUR_API_KEY"}

res = requests.post("https://web-production-9a18.up.railway.app/api/developer/generate-image", json=payload, headers=headers)
data = res.json()

# Response contains base64 encoded image
print(data["image_b64"])

Handling Errors

The API uses standard HTTP status codes to indicate success or failure.

CodeMeaning
200Success. The magic happened.
401Unauthorized. Invalid API Key or Firebase Token.
402Payment Required. Insufficient credits in wallet.
500Internal Server Error. Something broke on our end (or upstream AI provider).
