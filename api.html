Introduction

Welcome to the Elvion AI Developer API. Our platform provides developers with programmatic access to our state-of-the-art conversational and generative AI models. Whether you need a quick and efficient chatbot, a deep research assistant, or a powerful image generator, our API is designed to be simple, robust, and easy to integrate.

This documentation provides a comprehensive guide to help you get started. You'll find detailed endpoint references, runnable code examples in multiple languages, and guides for handling authentication and errors. All API access is managed through a credit-based system, which you can manage from your Developer Dashboard.

Authentication

All requests to the Elvion AI API must be authenticated using an API key. You can generate, manage, and revoke your API keys from the Developer Dashboard.

To authenticate a request, you must include your API key in the Authorization header of your request, formatted as a Bearer token. Requests without a valid key will be rejected with a 401 Unauthorized error.

Header Format

HeaderValue
AuthorizationBearer YOUR_API_KEY

Bobby Chat

POST /api/developer/bobby/chat 2 Credits

Access the Bobby model, a fast and cost-effective conversational AI. It is ideal for general-purpose chat, simple Q&A, and creative text generation where real-time web access is not required.

Body Parameters

ParameterTypeRequiredDescription
promptstringYesThe message or question you want to send to the model.
systemPromptstringNoA custom system prompt to guide the model's personality or behavior.

Seraphina Chat

POST /api/developer/seraphina/chat 5 Credits

Access the Seraphina model, our premium, web-grounded research assistant. Seraphina excels at providing accurate, up-to-date information with citations. This endpoint supports streaming for real-time responses.

Body Parameters

ParameterTypeRequiredDescription
messagestringYesThe message or question for Seraphina.
historyarrayNoAn array of previous messages to provide context. Each object must have role ("user" or "assistant") and content (string).
streambooleanNoDefaults to true. Set to false to receive the full response at once.

Image Generation

POST /api/developer/generate-image 15 Credits

Generate high-quality, unique images from a text prompt. You can choose between different models and styles to achieve your desired visual aesthetic.

Body Parameters

ParameterTypeRequiredDescription
promptstringYesA detailed description of the image you want to create.
modelstringNoThe generation model. Examples: "photoreal", "v6", "anime_core". Defaults to "v6".
use_minimaxbooleanNoDefaults to false. Set to true to use the MiniMax model instead of Thena.

Error Handling

The API uses standard HTTP status codes to indicate the success or failure of a request. If a request fails, the response body will be a JSON object containing a detail key with a human-readable error message.

Status CodeMeaning
200 OKThe request was successful.
401 UnauthorizedAuthentication failed. Your API key is missing, invalid, or revoked.
402 Payment RequiredYour account has insufficient credits to perform this action.
422 Unprocessable EntityThe request body is malformed or missing required parameters.
5xx Server ErrorAn unexpected error occurred on our end.

import requests
import json

API_URL = "https://web-production-9a18.up.railway.app/api/developer/bobby/chat"
API_KEY = "YOUR_API_KEY_HERE"

headers = {
    "Authorization": f"Bearer {API_KEY}",
    "Content-Type": "application/json"
}
data = {
    "prompt": "Write a short poem about space."
}
response = requests.post(API_URL, headers=headers, json=data)

if response.status_code == 200:
    print(json.dumps(response.json(), indent=2))
else:
    print(f"Error {response.status_code}: {response.text}")

// JavaScript (Node.js) Example
async function callElvionAPI() {
    const API_URL = 'https://web-production-9a18.up.railway.app/api/developer/bobby/chat';
    const API_KEY = 'YOUR_API_KEY_HERE';

    const headers = {
        'Authorization': `Bearer ${API_KEY}`,
        'Content-Type': 'application/json'
    };
    const body = JSON.stringify({
        prompt: 'Write a short poem about space.'
    });

    try {
        const response = await fetch(API_URL, { method: 'POST', headers, body });
        const data = await response.json();

        if (!response.ok) {
            throw new Error(data.detail || 'An error occurred');
        }
        console.log(data);
    } catch (error) {
        console.error('API Call Failed:', error.message);
    }
}

callElvionAPI();
